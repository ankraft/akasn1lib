int asnReadInteger ( long *  value,
int  length 
)

Read only a integer value w/o the tag and length.

Read only an ineteger value from the opened ASN.1 file, without the tag and the length. These had to be read prior. value is a pointer to the buffer to encode the value in. length is the number of bytes to read from the file.

Returns
If an error occures, the function returns -1, or the length of the encoded integer value otherwise.
Note
Restriction: The Integer has to be less than 2^31.
