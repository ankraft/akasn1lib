int asnIsEOC ( void  )

Check for End-Of-Contents in the opened ASN.1 file.

This function provides a way to check whether the next token in the opened ASN.1 file is an End-Of-Contents token EOC.

Returns
If an EOC token is the next in the file, 1 is returned, 0 otherwise.
