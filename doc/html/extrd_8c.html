Read an X.400 EXTERNAL body part from an ASN.1 file.

Read an X.400 EXTERNAL body part from an ASN.1 file. Only direct-reference (oid) and encoding (0, 1, 2) are supported.

Returns
If an error occures, the function returns -1, or the number of bytes read otherwise.
