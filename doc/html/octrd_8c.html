Read an octet string from an ASN.1-File (w/o the tag and length)

Read an octet string from the opened ASN.1 file (only the data). The tag and the length must have been read prior to the call of this function. value is the variable to fill with the result. length is the length of the data to be read. maxlen is the maximum number of bytes to be read. pc is a flag whether the encoding is primitive (asnPRIMITIVE ) or constructed (asnCONSTRUCTED ).

Returns
If an error occures, the function returns -1. Otherwise, the number of bytes read is returned.
Note
The maximum number of bytes has to be less than MAXINT.
