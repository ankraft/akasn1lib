int asnWriteLength ( long  length)

Write an encoded ASN.1 length field to the open ASN.1 file.

Encode and write an ASN.1 length field to the opened ASN.1 file. length is the length to be encoded.

Returns
If an error occures, the function returns -1, or the length of the encoding otherwise.
Note
Restriction: supports only length between 0 and MAXLONG.
