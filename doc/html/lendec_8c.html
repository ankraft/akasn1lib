int asnDecodeLength ( byte *  buffer,
long *  length 
)

Decode a "length" field.

This function decodes a "length" field of an ASN.1 value. buffer contains the encoded length, length is a pointer to a long variable which should get the result.

Returns
The function returns the length of the encoding in buffer.
Note
Restriction: supports only length between 0 and MAXLONG.
See also
asnEncodeLength, asnReadLength, asnWriteLength
