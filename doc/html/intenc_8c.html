int asnEncodeInteger ( byte *  buffer,
long  value 
)

Encode an integer value with tag and length.

Encode an Integer (C-type long) with tag and length into a buffer. buffer is the buffer to fill with the resulting encoding, value is the integer value to encode.

Returns
If an error occures, the function returns -1, or the length of the result in buffer otherwise.
Note
Restriction: The Integer has to be less than 2^31.
See also
