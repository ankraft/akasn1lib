Read a tag and a length from the opened ASN.1 file.

Read a tag and a length from the opened ASN.1 file. buffer is a pointer to the buffer for the read tag. length is a pointer to a variable, which will get the decoded length.

Returns
If an error occures, the function returns -1, or 0 otherwise.
