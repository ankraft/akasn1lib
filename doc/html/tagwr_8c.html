Encode and write a tag to an ASN.1 file.

Encode and write a tag to the opened ASN.1 file. clasS is the class of the tag (asnUNIVERSAL , asnAPPLICATION , asnCONTEXT , asnPRIVATE ). pc is a flag to indicate wether the data following the tag is constructed or primitive (asnPRIMITIVE , asnCONSTRUCTED ). tag is the number of the tag.

Returns
If an error occures, the function returns -1, or the size of the encoded tag otherwise.
