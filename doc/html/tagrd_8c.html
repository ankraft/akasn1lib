int asnReadTag ( byte *  buffer)

Read a tag from the opened ASN.1 file.

Read a tag from the opened ASN.1 file. buffer is a pointer to the buffer which will get the resulting tag encoding.

Returns
If an error occures, the function returns -1, or the size of the tag in buffer otherwise.
