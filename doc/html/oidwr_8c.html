int asnWriteOid ( char *  id)

Write an object identifier into an ASN.1 file.

Write an object identifier into an opened ASN.1 file. id is the enum value to be encoded. The format of the input must be "a b c ...".

Returns
If an error occures, the function returns -1, or the number of bytes written to the file otherwise.
Note
Restriction: Supports only subid's less or equal 16383.
