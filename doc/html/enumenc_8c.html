int asnEncodeEnum ( byte *  buffer,
long  value 
)

Encode an enumeration value with tag and length.

Encode an enumeration (C-type long) with tag and length into a buffer. This function does the same as the function asnEncodeInteger except the encoding value of the tag. buffer is the buffer to fill with the resulting encoding, value is the enumeration value to encode.

Returns
If an error occures, the function returns -1, or the length of the result in buffer otherwise.
See also
asnDecodeEnum, asnReadEnum, asnWriteEnum
Note
Restriction: The Integer has to be less than 2^31.
