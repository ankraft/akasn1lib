Decode an object identifier from a buffer into a string.

Decode an object identifier from buffer and return it in the format "a b c ...". buffer contains the encoded object identifier. length is the length of the encoding in buffer. id is a pointer for the resulting string.

Returns
The length of the result string id is returned.
Note
Restriction: Supports only subid's less or equal 16383.
