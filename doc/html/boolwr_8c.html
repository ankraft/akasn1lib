int asnWriteBoolean ( byte  value)

Write a boolean value into an ASN.1 file.

Write a boolean value into an opened ASN.1 file. value is the boolean value to be encoded. It must be either asnFALSE or asnTRUE .

Returns
If an error occures, the function returns -1, or 0 otherwise.
