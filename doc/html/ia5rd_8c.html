Read only the IA5-String w/o the tag and length.

Read only the IA5-String from the opened ASN.1 file, without the tag and the length. These had to be read prior. value is a pointer to the buffer to encode the string in. length is the number of bytes to read from the file. maxlen is maximum number of characters to be read.

Returns
If an error occures, the function returns -1. Otherwise, the number of characters read is returned.
Note
Restriction: The length of the string has to be less than MAXINT.
See also

Read an IA5-String from the opened ASN.1 file (with tag and length)

Read an IA5-String from the opened ASN.1 file, including the tag and the length of the encoding. value is the variable to fill with the result, maxlen is the maximum number of characters to be read (mostly the size of value).

Returns
If an error occures, the function returns -1. -2 is returned if the value read is not of type IA5-String. Otherwise, the number of characters read is returned.
Note
Restriction: The length of the string has to be less than MAXINT.
