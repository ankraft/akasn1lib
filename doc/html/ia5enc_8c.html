Encode an IA5-String with tag and length.

Encode an IA5-String with tag and length into a buffer. value is a 0-terminated C-string. buffer is a buffer in which the encoding of the tag and the length will be done. The length of the encoding in buffer will be returned in length.

Returns
If an error occures, the function returns -1, -2 of the value is not a legal IA5-String or the length of the result in buffer otherwise.
Note
Restriction: The length of the string has to be less than MAXINT.
See also
